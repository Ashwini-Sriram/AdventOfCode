# AdventOfCode


## How to start
Step 1 : Developer fork the repo
Step 2 : Update the day of question
Step 3 : Get the AOC_Token (Steps are enlisted here)
Step 4 : Put the AOC_Token in the .env file
Step 5 : Code your solution in solution.py
Step 6 : Run main.py
